## 제품 스펙 (절대 AI가 임의 작성 금지)
- PPF: 시리즈50(185μm,셀프힐링O,고광택O), 시리즈100(190μm,셀프힐링O,고광택O), 시리즈150(190μm,셀프힐링O,발수코팅O,고광택O), PRO200유광(200μm,셀프힐링O,발수코팅O,고광택O), PRO200무광(200μm,셀프힐링O,발수코팅O)
- 발수코팅 언급 규칙: 시리즈50·시리즈100의 발수코팅 여부는 블로그 글에서 절대 언급 금지. 발수코팅은 시리즈150, PRO200에서만 언급 가능
- 구조: 클리어코트+벌크우레탄+접착제 3층
- 컬러PPF: 3M PWF (건식+습식 가능)
- 랩핑: 3M 2080
- 썬팅: 3M 크리스탈라인(MOF 240겹), 루마 버텍스 900

## 시공 규칙
- PPF 외부: 습식만. 판시공 핸드커팅+안쪽마감. 오버랩 안함
- PPF 실내: 컴퓨터 재단. 물 최소화
- 랩핑/PWF: 열 성형+오버랩 사용
- 크롬죽이기: 3M 2080 글로스블랙. 미탈거(손잡이만 탈거)
- 썬팅: 이팀장님 1인시공. 하루 한대
- 작업자: PPF/랩핑/크롬죽이기=대표, 썬팅=이팀장님, 김막내=수동배정
- QC 2주, 메인터넌스 6개월
- 신차패키지 순서: 검수→썬팅→PPF/크롬죽이기→코팅

## 블로그 톤앤매너
- 매장에서 고객한테 설명하듯이
- 마침표 쓰지 말것
- "붙인다"가 아니라 "입힌다"
- PPF 시공 중 열 표현 금지, 오버랩 금지, 열 성형 단어 금지
- 패널별 나열 금지. 2~3곳만 깊이 설명

## 블로그 품질 규칙 (97점 기준)
- 고객 인용/행동 묘사 완전 차단: "고객님","오너분","말씀","하셨다","하셨어요","고민이 많으셨다","알아보시다가","문의가 많은데" 등 고객 행동/말 묘사 일체 금지. 대표 시점의 시공 철학·작업 소회로 대체
- 제목 규칙: "[지역명] [차종] [서비스] — [차별화 포인트]" 형식. 서초/강남/방배 중 하나 필수. SEO 키워드 3개 이상
- 문단 규칙: 모든 문단 최대 3줄 이하. 4줄 이상 반드시 분할. 줄바꿈 후 한 줄 공백
- SEO: 본문에 지역명 최소 2회, 차종+서비스 키워드 3회 이상, 시공 기간 정보 포함
- 마케팅: 기술적 차별점 강조(습식·핸드커팅·안쪽마감), 작업 철학으로 마무리
- 공백 제외 2000자 이상 필수

## CRM 구조
- 탭: 고객 목록 / 예약·일정 / 사후관리 / 상담 기록 / 매출 대시보드 / 재고 관리 / 견적 템플릿
- 담당자: 대표, 이팀장님, 김막내
- 유입경로: 카페, 블로그, 소개/재방문, 유튜브, 고정, 경로X, 기타

## 배포 규칙
- 코드 수정 후 반드시: 타입 체크(npx tsc --noEmit) → 빌드(npm run build) → 커밋+푸시 → npx vercel --prod
- .next 캐시 에러 시: rm -rf .next 후 재시도
- 견적서 URL: NEXT_PUBLIC_BASE_URL 환경변수 사용
